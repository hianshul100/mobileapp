# SkillForge Flutter User Stories

1. As a new learner, I want to create an account with a username, email, and password so I can save my progress.
2. As a returning learner, I want to log in with my email and password so I can access my saved learning dashboard.
3. As a learner, I want clear validation errors on signup so I can correct missing or invalid registration information.
4. As a learner, I want clear validation errors on login so I know when my credentials do not match an account.
5. As a learner, I want a home screen with recommended lessons and a visible app logo so I can quickly choose what to study.
6. As a learner, I want to open a lesson detail screen from the home screen so I can review lesson information.
7. As a learner, I want to favorite lessons and see them on my profile so my preferred lessons persist locally.
8. As a learner, I want the app to load fresh content from an external API so the app has dynamic data.
9. As a learner, I want settings and notification controls so I can customize reminders and receive test alerts.
